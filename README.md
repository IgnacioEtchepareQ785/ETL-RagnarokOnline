# Ragnarok Online ETL Pipeline

ETL pipeline que consume una API pública de Ragnarok Online para extraer
información de monsters y sus drops.

## Tecnologías
- Python
- requests
- pandas

## Arquitectura
API → Extract → Transform → Load → CSV

## Características
- Manejo de errores de API
- Validación de datos
- Código modular ETL
- Estructura profesional

## Uso
python main.py
